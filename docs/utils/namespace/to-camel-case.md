# toCamelCase

将命名转换为驼峰命名

## 使用

```js
import { toCamelCase } from '@pinos-ui/utils'
```

## 代码示例

<demo src="./demos/to-camel-case.vue" ></demo>

## 参数说明

| 参数名    | 说明   | 类型   | 必传   | 默认值  |
| ---- | ---- | ------ |  ------- |  ------- |
| value | 要转换的值 |  string |  是 |  -  |

## 类型声明

```js
export declare function toCamelCase(value: string): string;
```
